## 设计原则

- **移动端优先**: 所有界面优先考虑移动端体验
- **深色主题**: 默认深色背景 (#030303)
- **高对比度**: 使用荧光绿 (#B8FF00) 作为主色
- **简洁直观**: 减少不必要的装饰元素
- **安全区域**: 适配刘海屏和底部手势条
